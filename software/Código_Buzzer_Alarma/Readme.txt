Bien, como podéis ver uso el pin número 6 (cable azul para tierra, etiquetado como GROUND)
y el pin número 12 (cable rojo para el positivo, etiquetado como GPIO18). En mi caso conecto
el timbre directamente si ninguna resistencia. Si vuestro timbre requiere resistencia o no
estáis seguros os recomiendo  usar una resistencia de 470 omnios (colocadla en el cable
positivo antes del timbre). Si habéis puesto una resistencia y el timbre no suena o suena muy
bajo, sustituid la resistencia por una menor.
